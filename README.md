## LibrainX
官方主页:<http://im.librazy.org//*XTODO*/>  

### 概述
LibrainX是由Librazy也就是我（目前）独立开发的的开源网页解谜游戏，关卡不定期升级中~
绝对不会真的大概的确没有什么不坑爹的东西.
无需Flash支持.
开源且自由.
支持自动保存进度.
采用SHA512和AES加密,您就老老实实猜吧,别老想着破解了.
关卡密码格式参考本关提示。
